1. Open folder C:\Users\<USERNAME>\Documents\My Games\FarmingSimulator2022\modSettings\FS22_DashboardLive
2. Create a folder with the map name, e.g. "Elmcreek"
3. Inside the map folder place your custom minimap texture
4. Create a new file named "minimap_config.xml" and paste the following xml content into it:

<?xml version="1.0" encoding="utf-8" standalone="no"?>
<dashboardLive>
    <minimapConfig useCustomMinimap="true" filename="overview.dds"/>
</dashboardLive>

5. Change the value of filename to the name of your custom minimap texture file.
